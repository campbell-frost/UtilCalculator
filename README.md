# UtilCalculator

This project allows a user to input how much money goes to each utility per month, and calculates what each utility needs to give the others to make each ammount even.  

The use-case for this is each monthly utility is covered by a different person.  At the end of the day, each person wants to spend the same ammount of money.  This app lets the users input how much they spend per month and then returns how much needs to be payed to the people with the higher costs to make everything even
